# Java Basics

A structured collection of Java programs written during my MCA studies — covering core language concepts from OOP fundamentals to GUI programming and DSA problem solving.

## Topics Covered

### Object-Oriented Programming
Hands-on examples of the four OOP pillars — encapsulation, inheritance, abstraction, and polymorphism — applied to real class hierarchies like vehicles, students, and boxes.

Key files: `SingleInheritance.java`, `multiLevelExample.java`, `VehicleAbstract.java`, `VehicleInterface.java`, `DemoBoxWeight.java`

### Constructors & Static Members
Demonstrates the difference between instance constructors, static methods, and static initialisation blocks, and when to use each.

Key files: `ConstructorStaticMethodStaticBlock.java`, `ConstructorSum.java`

### Method Overloading & Object Passing
Shows how Java resolves multiple method signatures and how objects behave when passed as parameters (pass-by-reference of the reference).

Key files: `MethodOverloading.java`, `PassObjPara.java`

### String & Array Manipulation
Programs that search, count, modify, and process strings and arrays using built-in Java methods.

Key files: `LongestString.java`, `RemoveElement.java`, `CountLetterNum.java`, `StringMethods.java`

### Exception Handling
Custom exception classes demonstrating the difference between checked and unchecked exceptions, with test cases for negative input validation.

Key files: `NegativeInputTest.java`, `NegativeNumberException.java`

### AWT Calculator
A functional desktop calculator built using Java's AWT (Abstract Window Toolkit) — handles arithmetic operations with an event-driven GUI.

Key files: `CalculatorAWT.java`, `Calculator_AWT.java`

### DSA Problems
Select LeetCode-style problems solved in Java, covering linked lists, sliding window technique, and bit manipulation.

Key files: `winner_of_the_linked_list_game.java`, `min_size_subarray_sum.java`, `binNo_alt_bits.java`

## How to Run

```bash
# Compile any file
javac FileName.java

# Run it
java FileName
```

> Note: `.class` files are compiled bytecode. You can safely delete them — recompile from the `.java` source files anytime. A `.gitignore` excluding `*.class` is recommended.

## Skills Demonstrated

- OOP: encapsulation, inheritance, abstraction, polymorphism
- Constructors, static methods, and static blocks
- Method overloading and object passing
- String and array manipulation
- Custom exception handling
- Java AWT GUI programming
- DSA problem solving in Java
