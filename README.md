# Java Basics

A collection of Java programs covering core language concepts — written as hands-on practice during my MCA studies.

## What's Inside

This repo is organised around the key pillars of Java programming:

### Object-Oriented Programming
Practical examples of OOP in action — classes, objects, inheritance (single and multi-level), encapsulation, abstract classes, and interfaces.

Key files: `SingleInheritance.java`, `multiLevelExample.java`, `VehicleAbstract.java`, `VehicleInterface.java`, `DemoBoxWeight.java`

### Constructors & Static Members
Demonstrates the difference between instance constructors, static methods, and static initialisation blocks.

Key files: `ConstructorStaticMethodStaticBlock.java`, `ConstructorSum.java`

### Method Overloading & Object Passing
Shows how Java handles multiple method signatures and how objects are passed as parameters.

Key files: `MethodOverloading.java`, `PassObjPara.java`

### String & Array Manipulation
Programs that search, count, modify, and process strings and arrays.

Key files: `LongestString.java`, `RemoveElement.java`, `CountLetterNum.java`, `StringMethods.java`

### Exception Handling
Custom exception classes and examples of checked vs unchecked exceptions.

Key files: `NegativeInputTest.java`, `NegativeNumberException.java`

### AWT Calculator
A desktop calculator built using Java's AWT (Abstract Window Toolkit) library.

Key files: `CalculatorAWT.java`, `Calculator_AWT.java`

### DSA Problems (Java)
Select LeetCode-style problems solved in Java — linked lists, sliding window, bit manipulation.

Key files: `winner_of_the_linked_list_game.java`, `min_size_subarray_sum.java`, `binNo_alt_bits.java`

## Running Any Program

```bash
# Compile
javac FileName.java

# Run
java FileName
```

> Note: `.class` files are compiled bytecode included for reference. You can safely delete them and recompile from the `.java` source files.

## Skills Covered

- Object-Oriented Programming (OOP)
- Inheritance, Encapsulation, Abstraction, Interfaces
- Constructors and static members
- Method overloading and object passing
- String and array manipulation
- Custom exception handling
- Java AWT GUI programming
- Basic DSA problem solving
